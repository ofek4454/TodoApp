# TodoApp
Todo application using React for front-end and Python for back-end.

## Features:
- add new Todo to the list.
- mark Todo as completed/uncompleted.
- tabs for all/active/completed Todos.
- delete todos.

### Run server:
```
cd todo_server
poetry run uvicorn main:app --reload
```
### Run front-end:
```
cd todo_app
npm start
```
